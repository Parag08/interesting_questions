# Review system

A review system made using nodejs

## Pre-requisites

Mongo server running on 127.0.0.1:27017
nodeJS version: v10.16.3
npm version: 6.9.0

## How to run

```
git clone https://github.com/Parag08/interesting_questions.git
cd interesting_questions/casaone
npm install
node src/index.js
```
